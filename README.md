
This repository contains KiCad parts used in our devices

# usage
1. pull the repository
2. open KiCad
3. open menu -> Preferences -> Manage Symbol Libraries
4. hit the [ğŸ“] button at the bottom, navigate to the pulled directory, and select all *.kicad_sym files, confirm by [Open], then [OK]
5. open menu -> Preferences -> Manage Footprint Libraries
6. hit the [ğŸ“] button at the bottom, navigate to the pulled directory, select it by clicking [Open], then [OK]
7. restart KiCad
